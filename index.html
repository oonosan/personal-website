Simpletron

Simpletron es un programa hecho en C para la materia Algoritmos y Programación de la carrera de Ingeniería Electrónica.

El programa está hecho para que sea ejecutado desde la terminal de Linux. El mismo recibe un archivo que contiene "palabras" (dígitos con signo) en formato Big Endian, cada palabra indica una operación a realizar. El resultado de las operaciones realizadas se exportan a un archivo binario o de texto.

Tanto el código como su ejecución y el enunciado del trabajo práctico se encuentran en Github
